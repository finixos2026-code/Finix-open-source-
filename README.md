
# Finix Open Source

Welcome to Finix Open Source!

Finix Open Source is a collection of open-source projects created and maintained by Alkha. The goal of this initiative is to learn, share knowledge, and build useful software and hardware projects for everyone.

## Vision

To create accessible, educational, and open-source technologies in various fields including:

- Cloud Computing
- Internet of Things (IoT)
- Robotics
- Embedded Systems
- Software Development
- Operating Systems
- Utilities and Libraries

## Projects

### Finix Cloud
A lightweight cloud storage platform built with Node.js.

Features:
- Multi-user support
- File upload and management
- Storage quota system
- File preview
- Security protections

Status: Active Development

---

### Finix IoT
IoT experiments and platforms based on ESP32, Arduino, and other microcontrollers.

Status: Planning / Development

---

### Finix Robotics
Robotics projects, automation systems, and embedded control experiments.

Status: Planning

---

### Finix Converter
Utility tools for data and file conversion.

Status: Development

---

### Finix OS
Operating system and low-level system experiments.

Status: Research
